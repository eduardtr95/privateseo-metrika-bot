# Как помочь проекту

Перед большой доработкой откройте Issue и кратко опишите пользовательскую проблему. Для небольшого исправления можно сразу отправить Pull Request.

```bash
python3 -m venv .venv
.venv/bin/pip install -e '.[dev]'
.venv/bin/pytest -q
```

Новые правила аналитики должны сопровождаться тестами на малые числа, нулевой предыдущий период и обычный рабочий сценарий. Не добавляйте в тесты реальные токены, номера счётчиков и данные сайтов.
