# Политика обработки данных

PrivateSEO Аналитика использует данные Яндекс Метрики только для формирования отчётов по запросу пользователя и по выбранному им расписанию.

Сервис хранит:

- Telegram chat ID и username, если он передан Telegram;
- выбранный счётчик и идентификаторы выбранных целей;
- OAuth access/refresh-токены в зашифрованном виде;
- технические события: подключение, выбор счётчика, формирование отчёта.

Сервис не хранит сырые выгрузки Яндекс Метрики, содержимое сформированных отчётов и персональные данные посетителей сайтов.

Команда `/disconnect` удаляет OAuth-токены и настройки Метрики. Команда `/delete_me` удаляет все данные пользователя из базы бота. Пользователь также может отозвать доступ приложения в настройках Яндекс ID.

Самостоятельно развёрнутые копии работают под ответственностью их владельцев; эта политика относится к официальному экземпляру PrivateSEO.
